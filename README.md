# TicTacToe - React

Features implemented:
- 3x3 TicTacToe board
- Players alternate X then O
- Prevent overwriting cells
- Win & draw detection
- Status text: Next Player / Winner / Draw
- Clean responsive UI + hover effects

State management:
- Context + `useReducer` (see `src/game/GameContext.jsx` and `src/game/reducer.js`)

Manual feature (required):
- Undo Move (button)

Advanced feature:
- Move History + Time Travel (click a move to jump to it)

Run locally:

```bash
npm install
npm run dev
```

Build for production:

```bash
npm run build
npm run preview
```
